hexo 备忘

head 参数

  • title 文章的标题
  • date 创建日期 （文件的创建日期 ）
  • updated 修改日期 （ 文件的修改日期）
  • comments 是否开启评论 true
  • tags 标签
  • categories 分类
  • permalink url中的名字（文件名）

常用命令

hexo clean 删除本地静态文件（Public目录），可不执行。
hexo g 生成本地静态文件（Public目录）
hexo d 将本地静态文件推送至github（hexo d）
